# 概要
料理を自由に投稿できるサービスです。

子供から大人まで、幅広い利用者を想定しています。

## ターゲット
- 小学生
- 中高生
- 大学生
- 社会人

## 使い方
1. ユーザー登録
    - リスト
1. メール送信
1. 写真投稿
1. 承認ボタン 

| URL | サーブレット|JSP|
|:---:|:---:|:---:|
|/home|HomeServlet|home.jsp|
|/about|AboutServlet|about.jsp|
